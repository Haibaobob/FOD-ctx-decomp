# FOD_ctx_decomp
Source code for MICCAI 25 paper "GPU Accelerated Modeling of Cortical Radial and Tangential Connectivity Changes in Neurodegeneration"

## Getting Started
download the example data from `https://drive.google.com/drive/folders/1FQY8dK7keOrCHOaKCsWNTL1pnDcoeBUI?usp=sharing`

unzip `tar -zxvf example.tar.gz`

run `example_usage.ipynb`
